# CV Creator

A modern, reactive web application for building and creating professional CVs/Resumes. Built with React, TypeScript, and Tailwind CSS.

## Features

- **Real-time Preview**: See your changes instantly as you edit.
- **Comprehensive Sections**:
  - Personal Information (including social links)
  - Professional Experience
  - Education
  - Skills
  - Languages
- **PDF Export**: Easily download your CV as a high-quality PDF.
- **Auto-save**: Your progress is automatically saved to your browser's local storage, so you don't lose your work.
- **Modern UI**: Clean and intuitive interface designed with Tailwind CSS.

## Tech Stack

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) (Icons)
- [React-to-Print](https://github.com/gregnb/react-to-print)

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd cv-creator
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to start editing.

### Build

Build the application for production:
```bash
npm run build
```

This will generate a `dist` folder ready for deployment.

## Usage

1. **Edit**: Fill in your details in the Editor section on the left.
2. **Preview**: View the layout on the right.
3. **Save**: Auto-save is enabled by default. You can toggle it in the top navigation bar.
4. **Download**: Click the "Download PDF" button to save your CV.
